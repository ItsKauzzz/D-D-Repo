# Icon assets

Esta pasta guarda os caminhos de ícones usados pelo mapa.

Os arquivos de imagem (PNG) **não são versionados** neste repositório.
Adicione localmente os PNGs por tipo para uso em runtime, por exemplo:

- `Icon/Cidade/icon_1.png`
- `Icon/Vila/icon_1.png`
- `Icon/Acontecimento/icon_1.png`

Se não existir arquivo de ícone, o mapa oculta a miniatura automaticamente.
